# Weather Dashboard
## Purpose
A dashboard that uses OpenWeather API integration to retrieve a 5 day forecast using the city name entered in the search function.

## Built with
* HTML
* CSS
* JavaScript
* JQuery
* Moment.js
* Bootstrap
